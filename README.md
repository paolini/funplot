## Development

run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy

Build the optimized production site with:

```
npm run build
```

then run a web server on the `out` directory.
